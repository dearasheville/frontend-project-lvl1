[![Actions Status](https://github.com/dearasheville/frontend-project-lvl1/workflows/hexlet-check/badge.svg)](https://github.com/dearasheville/frontend-project-lvl1/actions)
[![Maintainability](https://api.codeclimate.com/v1/badges/3c7e47867ca48a511617/maintainability)](https://codeclimate.com/github/dearasheville/frontend-project-lvl1/maintainability)

A set of six mini-games.

## Installation
```sh
$ make install
```

### brain-even
Answer whether the number is even.
```sh
$ brain-even
```
[![Asciicast brain-even](https://asciinema.org/a/QLxpGDbC4zuqBteuqsM6IfZMs.svg)](https://asciinema.org/a/QLxpGDbC4zuqBteuqsM6IfZMs)

### brain-calc
Calculate simple mathematical operations.
```sh
$ brain-calc
```
[![Asciicast brain-calc](https://asciinema.org/a/flZf5CLYDAUohcn6zdTrmrcxu.svg)](https://asciinema.org/a/flZf5CLYDAUohcn6zdTrmrcxu)

### brain-gcd
Make the greatest common divisor of two numbers.
```sh
$ brain-gcd
```
[![Asciicast brain-gcd](https://asciinema.org/a/ov2UvUeo0wxybWi52FNmNJIcR.svg)](https://asciinema.org/a/ov2UvUeo0wxybWi52FNmNJIcR)

### brain-progression
Complete the algebraic sequence with the missing number.
```sh
$ brain-progression
```
[![Asciicast brain-progression](https://asciinema.org/a/NsXf7ynYcZs9C23GpmKHvhwpg.svg)](https://asciinema.org/a/NsXf7ynYcZs9C23GpmKHvhwpg)

### brain-prime
Answer whether the number is a simple.
```sh
$ brain-prime
```
[![Asciicast brain-prime](https://asciinema.org/a/b6yq9myLEnSDm5tkVsaAMfVWo.svg)](https://asciinema.org/a/b6yq9myLEnSDm5tkVsaAMfVWo)
